flex知识点

使用flex布局第一步需要先设置父元素为flex布局 display:flex

  1. flex布局父项常见属性：

    属性 描述
    flex-direction 设置主轴方向
    justify-content 设置主轴上子元素的对齐方式
    flex-wrap 设置子元素是否换行
    align-content 设置侧轴上的子元素的对齐方式(多行)
    align-items 设置侧轴上的子元素的排列方式（单行）
    flex-flow 复合属性，相当于同时设置flex-direction和flex-wrap

  2. flex布局子项常见属性

    属性 描述
    flex 控制子元素所占的份数
    align-self 控制单个子元素在侧轴的排列方式
    order 定义子元素的排列顺序

flex布局父项常见属性

flex-direction主轴方向

flex-direction属性决定主轴的方向，即子元素的排列方向

默认主轴方向为 **x轴 **

属性值 说明
row 默认值，从左到右排列即x为主轴
row-reverse 从右到左排列
column 设主轴为y轴，从上到下排列
column-reverse 从下到上排列

justif-content主轴上的对齐方式

justify-content属性设置子元素在 主轴 上的对齐方式

属性值 说明
flex-start 默认值，左对齐；若主轴是y轴就是上对齐
flex-end 右对齐；若主轴是y轴就是下对齐
center 在主轴上居中对齐
space-around 平分空间，平均排列
space-between 第一个与最后一个子元素贴边，其余平分空间

flex-wrap是否换行

flex-wrap属性决定子元素是否换行，默认情况下不换行，若轴线上装不下则会缩小子元素的宽度

属性值 说明
nowrap 默认值，不换行
wrap 换行

align-items（单行）侧轴上的对齐方式

align-items属性设置子元素在 侧轴 上的对齐方式，一般配合justify-content属性使用

属性值 说明
flex-start 默认值，若当前主轴是x轴，则在y轴上 上对齐
flex-end 若当前主轴是x轴，则在y轴上 下对齐
center 若当前主轴是x轴，则在y轴上 垂直居中
stretch 拉伸子元素，占满父元素前提是子元素没有高度

align-content(多行) 侧轴上的对齐方式

align-items属性设置子元素在 侧轴 上的对齐方式，并且只能用于子元素出现换行的情况（多行），在当行下无效果

属性值 说明
flex-start 默认值，若当前主轴是x轴，则在y轴上 上对齐
flex-end 若当前主轴是x轴，则在y轴上 下对齐
center 若当前主轴是x轴，则在y轴上 垂直居中
space-around 子元素在侧轴平分空间
space-betwe 子元素在 侧轴贴边，其余平分空间
stretch 设置子元素平分父元素高度，前提子元素没有高度

flex-flow 复合属性

同时设置主轴方向（flex-direction）和是否自动换行（flex-wrap）

flex-direction:column;
flex-wrap:wrap;
/*等于*/
flex-flow:conlumn wrap;

flex布局子项常见属性

flex属性(重要)

flex属性定义子元素分配剩余空间的份数

.item{
    flex:<number>; /* default=0 */
}

align-self 设置单个子元素的对齐

align-self控制单个子元素在侧轴的排列方式，可以覆盖align-items。

默认值为auto表示继承父元素的align-items属性，若没有父元素，则等同于stretch

属性值 说明
flex-start 默认值，若当前主轴是x轴，则在y轴上 上对齐
flex-end 若当前主轴是x轴，则在y轴上 下对齐
center 若当前主轴是x轴，则在y轴上 垂直居中
stretch 拉伸子元素，占满父元素前提是子元素没有高度

order控制单个子元素排序

order属性控制单个子元素的排列位置

.item{
    order:<number>; /* default=0，越小越靠前 */
}
